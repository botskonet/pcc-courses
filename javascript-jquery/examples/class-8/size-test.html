Testing how viewport width affects media queries in different browsers

When the viewport is wider than 750px, the background should be white. When the viewport is 750px or narrower, the body should get a pink background. The ruler at the top of the page reveals that this happens at different widths in WebKit browsers. For details, see the blog post Media queries, viewport width, scrollbars, and WebKit browsers.
